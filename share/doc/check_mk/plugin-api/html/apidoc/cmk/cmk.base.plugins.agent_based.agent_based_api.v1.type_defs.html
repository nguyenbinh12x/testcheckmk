agent_based_api.v1.type_defs¶

This module contains type definitions that users can use if they choose to leverage the power of type annotations in their check plugins.

Example

For a parse function that creates a dictionary for every item, for instance, you could use

>>> from typing import Any, Mapping
>>>
>>> def parse_my_plugin(string_table: StringTable) -> Mapping[str, Mapping[str, str]]:
...     pass
>>>
>>> # A check function handling such data should be annotated
>>> def check_my_plugin(
...     item: str,
...     params: Mapping[str, Any],
...     section: Mapping[str, Mapping[str, str]],
... ) -> CheckResult:
...     pass
